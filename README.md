# make-Xcode-colorful

Let's make Xcode colorful

# How to

1. Install **UbuntuMono** family in your system.
2. Copy/paste **Dracula.xccolortheme** file to xcode theme folder, in case there is no such folder, create one by:
    ```
    mkdir -p ~/Library/Developer/Xcode/UserData/FontAndColorThemes
    ```
3. Open xcode, select Dracula theme in Preferences. Enjoy! This works best with xcode darkmode.
